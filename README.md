LabSix: Student Grade Processing

A Java file-processing lab that reads student grade data from a text file and uses it to practice parsing and reporting.

Highlights
Reads from studentGrades.txt.
Practices file input with Scanner.
Processes structured grade data.
Prints results through the console.

Overview
LabSix uses a grade file as the source of program data. The program reads the file, processes the contents, and prints information based on the student records.
The lab focuses on combining file I/O, variables, loops, and basic data processing.

Why This Project
This project demonstrates how Java programs can work with records stored outside the source code. It is a practical foundation for later assignments involving CSVs, databases, and larger datasets.

Usage
Compile: Save the code as LabSix.java and compile using javac LabSix.java.
Run: Execute the compiled code with java LabSix.
Keep studentGrades.txt in the same folder as LabSix.java.

Installation
Ensure you have the Java Development Kit (JDK) installed on your system.
Open a terminal in the lab folder before compiling so the program can find any required local text files.

Architecture
File Structure: The project consists of LabSix.java and studentGrades.txt.
Components: LabSix.java contains the parser and reporting logic. studentGrades.txt provides the grade input data.

Future Improvements
Add stronger handling for missing or malformed grades.
Calculate additional statistics such as min, max, and average.
Format the output as a cleaner grade report.
